Personal Projects Overview

**1. Clap-on clap-off light switch (ATtiny85)** I programmed an ATtiny85 using an Arduino as an ISP, soldered everything onto a protoboard, and built it into a modified electrical housing with an external gain knob to adjust microphone sensitivity. You plug a lamp in, plug it into the wall, it works.

**2. [Portable weather satellite image station](https://github.com/IsaacSavona/Weather-Satellite-Image-Station)** A friend and I built a device that intercepts live transmissions from NOAA and Russian Meteor weather satellites and processes them into images of earth. The setup was a software defined radio timed with cron jobs to activate when a satellite was passing overhead, a Raspberry Pi running Python decoding software, and a 3D printed compliant mechanism housing that held the battery, Pi and SDR together, which then friction-fit onto a collapsible PVC antenna mast you could take apart for transport. The whole thing ran autonomously in a field clearing. What I like about this one is that every layer required something different: RF signal processing, embedded Linux, mechanical design, fabrication. And it worked on the first field test. It became my undergrad thesis and I presented it at Dartmouth's Convergence Symposium.

**3. Music** I play saxophone and guitar. I've put out two releases, a [jazz record](https://www.youtube.com/@littlefriendsquartet4460) with two friends (a good chunk of the tracks are in asymmetric meters like 15/8, with improvised sections, though it's a bit contentious to call it jazz for that reason) and an [electronic/indie album](https://open.spotify.com/album/6yu7b2m1mCpDvLsOCI5nY7?si=Ahb0L-dOR6WCuWo9Yv9Q0Q) made in Delft where I contributed saxophone parts used as drone textures and harmonics. Less directly relevant but it's where I spend a lot of the rest of my time.

**4. [Active Inference / KL control demo](https://github.com/IsaacSavona/Active-Inference-Exploration)** I've been reading Active Inference by Karl Friston in a book club with a friend and built a Python simulation demonstrating KL control, a formalism adjacent to the book's core framework. Smaller project but it's the kind of thing I do in my own time: find a controls idea I don't fully understand and implement it until I do.
